2012年9月29日 Studio Kura電子工作室ワークショップ
===============================================

光るおもちゃを作ります。
ArduinoでいくつかのLEDを好きなパターンで点滅させて、
ぬいぐるみなどに入れます。

ここにあるサンプルのプログラムを改造して、
自分の好きなパターンをどうぞ作ってみてください。
